# Telegram News Collector

אתר Next.js שאוסף הודעות חדשות מערוצי טלגרם ומציג עמוד נפרד לכל ערוץ.

## איך זה עובד

האתר משתמש ב־Telegram Bot API (`getUpdates`) כדי לקרוא הודעות שהבוט קיבל
מערוצים שבהם הוא הוגדר כמנהל. כל בקשה לעמוד ערוץ מחזירה את ההודעות האחרונות
שנמצאו בעדכונים שהבוט קיבל.

> **הערה:** ה־Bot API מספק רק הודעות שהבוט קיבל לאחר שהצטרף לערוץ.

## התקנה מקומית

```bash
npm install
cp .env.example .env
npm run dev
```

הגדר את משתני הסביבה ב־`.env`:

- `TELEGRAM_BOT_TOKEN` – הטוקן של הבוט (אל תעלה לריפו).
- `TELEGRAM_CHANNELS` – רשימת ערוצים מופרדת בפסיקים, לדוגמה: `@channel1,@channel2`.
- `NEXT_PUBLIC_BASE_URL` – אופציונלי. כתובת הבסיס (למשל `http://localhost:3000`). אם לא מוגדר, השרת ינסה להרכיב כתובת מה־headers בזמן ריצה.

## פריסה ל־Vercel

1. צור פרויקט חדש ב־Vercel והעלה את הריפו.
2. הוסף את משתני הסביבה שהוזכרו למעלה דרך Dashboard של Vercel.
3. דאג שהבוט מחובר לערוצים הרלוונטיים עם הרשאות מנהל.
